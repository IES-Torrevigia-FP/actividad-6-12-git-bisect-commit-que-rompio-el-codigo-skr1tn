# Checkpoint 4 - git bisect run (parte opcional)

## Contenido de test_contador.sh

#!/bin/bash
# test_contador.sh - Script para git bisect run
# Retorna 0 (good) si python contador.py 5 imprime 6
# Retorna 1 (bad)  si imprime cualquier otro valor

python contador.py 5 | grep -qx "6"

## Explicación del script

El script usa grep -qx "6" para comprobar si la salida es exactamente "6".
- Código de salida 0: grep encontró "6" -> commit GOOD
- Código de salida 1: grep no encontró "6" -> commit BAD

## Comandos ejecutados

git bisect start
git bisect bad                # HEAD = a984b03
git bisect good e0d5ec7       # Versión inicial correcta
chmod +x test_contador.sh
git bisect run ./test_contador.sh
